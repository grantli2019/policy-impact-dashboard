# 策查查 — 产品设计文档

## 一、产品定位

**策查查**是一款面向普通人的政策决策辅助工具，核心 Slogan："读懂政策，做对决策"。

对标产品：企查查（企业信息查询）。策查查做的是"政策对人的影响"，而非"政策文本本身"。

---

## 二、目标用户画像

### 第一层：高频刚需用户（核心层，约 20%）

| 用户类型 | 典型场景 | 核心痛点 |
|---|---|---|
| 正在买房/换房的人 | 限购政策、公积金、房贷利率、退税，每一个都是几十万级别决策 | 信息碎片化：政策散落在住建局、税务局、公积金中心等不同网站 |
| 孩子即将入学的家长 | 学区政策、双减、多校划片，直接影响孩子教育路径 | 政策变化快：学区划分年年调整，信息来源不可靠 |
| 创业者/个体户 | 税收优惠、行业监管、营商环境，直接影响生意存亡 | 合规成本高：不知道有哪些优惠政策可以享受 |

### 第二层：阶段驱动用户（主力层，约 50%）

| 用户类型 | 典型场景 | 核心痛点 |
|---|---|---|
| 刚毕业的年轻人 | 社保怎么交、公积金有什么用、租房补贴怎么领 | 信息盲区大：根本不知道有哪些跟自己相关的政策 |
| 准备结婚生子的人 | 生育津贴、产假政策、首套房资格 | 时间窗口紧迫：错过申领期就无法补救 |
| 中年职场人 | 个税优化、养老金规划、职业转型 | 选择太多无法判断：不知道该优先关注哪条政策 |
| 临近退休的人 | 延迟退休、医保、长期护理保险 | 政策看不懂：官方文件术语太多，无法理解对自己意味着什么 |

### 第三层：信息敏感型用户（扩展层，约 30%）

| 用户类型 | 典型场景 | 核心痛点 |
|---|---|---|
| 投资者 | 政策风向对行业的影响 | 信息滞后：看到新闻时市场已经反应过了 |
| 财经/法律从业者 | 快速理解政策对客户的影响 | 缺乏结构化视角：需要快速把政策翻译成专业建议 |
| 关心时事的普通公民 | "新闻联播说的跟我有什么关系" | 新闻看不懂：LPR下调25个基点意味着月供省多少？ |

## 三、四大使用场景与痛点

### 场景 A：决策倒计时（最高频，最有价值）

> "我下周就要签合同买房了，公积金能贷多少？换房退税来不来得及？"

- **痛点**：信息碎片化 — 政策散落在不同政府网站，普通人要花数小时才能拼出完整信息
- **价值**：一个搜索框 + 场景化推荐，5分钟搞定决策

### 场景 B：人生转折点（低频但高价值）

> "我刚结婚/刚生孩子/刚换工作/准备退休，不知道有哪些政策跟我有关。"

- **痛点**：不知道自己不知道 — 最大的痛点，多数人根本不知道该去查什么
- **价值**：人生雷达消除认知盲区，自动扫描出机会、盲区和风险

### 场景 C：日常信息焦虑（中频中价值）

> "今天新闻说XX政策要变了，跟我有关系吗？"

- **痛点**：新闻看不懂 — 专业术语太多，无法理解对个人的影响
- **价值**：把政策翻译成"人话"，每条政策都有通俗解读和影响评分

### 场景 D：长期规划（低频高价值）

> "我35岁，该怎么规划未来10年的社保、房产和养老？"

- **痛点**：缺乏系统性视角 — 六大维度之间存在交叉影响，没人帮你看全局
- **价值**：六维度总览 + 政策关系图谱 + 政策日历，系统性规划

## 四、核心价值主张

策查查提供的不是"政策数据库"，而是**决策辅助**。

| 传统方式 | 策查查方式 | 价值差 |
|---|---|---|
| 自己去政府网站找文件 | 搜索即得，评分排序 | **省时间** — 从数小时到5分钟 |
| 看新闻标题自己猜影响 | 专业评分 + 通俗解读 | **降门槛** — 从看不懂到秒懂 |
| 只知道眼前的事 | 人生雷达扫描盲区 | **扩认知** — 消除"不知道自己不知道" |
| 凭感觉做决策 | 计算器 + 决策模拟器 | **算清楚** — 把政策翻译成具体数字 |
| 政策变了才知道 | 政策监控 + 每周更新 + 政策日历 | **提前知** — 主动推送变化 |

## 五、用户价值链路

```
🔍 搜索框（场景化入口）
    ↓
📡 人生雷达（主动发现盲区）
    ↓
📋 政策库（六维度结构化浏览）
    ↓
📄 政策详情（评分 + 通俗解读）
    ↓
🧮 工具箱（房贷/退税/资格/省息计算器）
    ↓
📅 政策日历（关键日期提醒）
    ↓
🎯 场景专题（完整决策指南）
```

## 七、产品功能架构

| 模块 | 功能 | 解决的问题 |
|---|---|---|
| 场景化搜索 | 输入"我要买房"自动关联一揽子政策 | 信息碎片化 |
| 人生雷达 | 6个阶段 × 6维度扫描，输出机会/盲区/风险 | 认知盲区 |
| 行动追踪 | 行动清单可标记完成，进度可视化 | 从知道到做到 |
| 复合画像 | 支持"上班族×购房者"等多身份组合 | 身份复杂性 |
| 政策日历 | 关键截止日期时间线，倒计时提醒 | 时间窗口紧迫 |
| 浏览历史 | 自动记录浏览过的政策 | 信息回溯 |
| 实用工具 | 房贷计算、退税计算、资格自测、公积金省息 | 把政策翻译成数字 |
| 政策监控 | 关键词订阅 + 每周更新推送 | 信息滞后 |

## 八、视觉规范

- **品牌名称**：策查查
- **主色调**：信任蓝 `#1677ff`（对标企查查）
- **辅助色**：成功绿 `#27ae60`、警告橙 `#e67e22`、风险红 `#e74c3c`、紫色 `#722ed1`
